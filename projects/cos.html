Cafeteria Ordering System (COS) Web App

This full-stack web application was built to simulate a cafeteria-style ordering experience. As project leader, I guided a team through the planning, development, and testing phases. Users can browse a menu, add items to a cart, and proceed to checkout through a simple, responsive UI.

Technologies Used: Python, Flask, HTML, CSS, JavaScript
